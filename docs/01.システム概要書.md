## 2. システム概要

### 2.1 目的
PayTrackは、入金・支払の予定および実績を管理し、抜け漏れ防止と業務効率化を実現するシステムである。

### 2.2 特徴
- 入出金の予定と実績を一元管理
- 繰り返し設定による自動生成
- 非エンジニアでも使えるUI

---

## 3. 技術スタック
- フロント: Next.js
- バックエンド: FastAPI
- DB: PostgreSQL
- インフラ: Docker / AWS想定
